<action>
Create `src/unitree_g1_dex3_stack-main/scripts/apriltag_detector_node.py` containing a single `AprilTagDetectorNode(rclpy.node.Node)` class.

Required structure (in order):

1. Shebang `#!/usr/bin/env python3` and a docstring summarizing the node (one paragraph: detect → filter → offset → TF → publish + optional OpenCV viz).

2. Imports — group in order:
   - stdlib: `collections`, `time`
   - numpy as np
   - cv2
   - rclpy + `rclpy.node.Node`
   - `rclpy.qos.qos_profile_sensor_data`
   - `rclpy.duration.Duration`
   - sensor_msgs.msg: `Image`, `CameraInfo`
   - geometry_msgs.msg: `PoseStamped`
   - cv_bridge: `CvBridge`
   - tf2_ros (full module — for Buffer, TransformListener, TransformException)
   - `import tf2_geometry_msgs  # noqa: F401` (CRITICAL — side-effect import; registers PoseStamped/Pose plugin)
   - pupil_apriltags: `Detector`
   - scipy.spatial.transform.Rotation as R (for matrix→quaternion)

3. Class `AprilTagDetectorNode(Node)` with `__init__(self)`:
   - `super().__init__('apriltag_detector')` — name MUST match YAML key.
   - declare_parameter calls for all 11 YAML fields per D-08, plus `imshow` (bool, default True).
   - Read into `self.<field>` instance variables. For `offset_xyz`, read as list and convert to a `numpy.ndarray` of shape (3,).
   - Construct `self.detector = Detector(families=self.tag_family, nthreads=1, quad_decimate=1.0, refine_edges=1)`.
   - Initialize `self.bridge = CvBridge()`.
   - Initialize `self.tf_buffer = tf2_ros.Buffer()` and `self.tf_listener = tf2_ros.TransformListener(self.tf_buffer, self)`.
   - Initialize `self.camera_params = None` (filled on first CameraInfo).
   - Initialize `self.dist_coeffs = None` (filled on first CameraInfo, or zeros if D is empty/zero).
   - Initialize `self.frame_times = collections.deque(maxlen=30)`.
   - Initialize `self.last_warn_time = 0.0` (for warn throttling).
   - Initialize `self.imshow_enabled = self.imshow` (mutable so 'q' key disables it).
   - If `self.imshow_enabled`: `cv2.namedWindow(self.window_name, cv2.WINDOW_NORMAL); cv2.resizeWindow(self.window_name, 640, 480)`. Window name format: `f"AprilTag detector (id={self.target_tag_id})"`.
   - Subscriptions:
     - `self.create_subscription(Image, self.rgb_topic, self.image_cb, qos_profile_sensor_data)`
     - `self.create_subscription(CameraInfo, self.camera_info_topic, self.info_cb, qos_profile_sensor_data)`
   - Publishers (default reliable QoS, depth 10):
     - `self.tag_pose_pub = self.create_publisher(PoseStamped, self.tag_pose_topic, 10)`
     - `self.target_pose_pub = self.create_publisher(PoseStamped, self.target_pose_topic, 10)`
   - Log a startup banner: `self.get_logger().info(f"AprilTag detector ready: family={self.tag_family} target_id={self.target_tag_id} tag_size={self.tag_size}m output_frame={self.output_frame} imshow={self.imshow_enabled}")`.

4. `info_cb(self, msg: CameraInfo)`:
   - On first call (when `self.camera_params is None`), extract `fx=msg.k[0], fy=msg.k[4], cx=msg.k[2], cy=msg.k[5]`.
   - Cache `self.camera_params = (fx, fy, cx, cy)`.
   - Cache `self.dist_coeffs = np.zeros(5, dtype=np.float64)` if `msg.d` is empty or all zeros, else `np.array(msg.d, dtype=np.float64)`.
   - Log once: `f"CameraInfo cached: fx={fx:.2f} fy={fy:.2f} cx={cx:.2f} cy={cy:.2f}"`.

5. `image_cb(self, msg: Image)`:
   - Update FPS: `self.frame_times.append(time.monotonic())`.
   - If `self.camera_params is None`: throttled warn "Waiting for CameraInfo on {topic}..." (throttle 5 s) and `return`.
   - Convert: `bgr = self.bridge.imgmsg_to_cv2(msg, desired_encoding='bgr8')`. Wrap in try/except CvBridgeError → throttled warn → return.
   - `gray = cv2.cvtColor(bgr, cv2.COLOR_BGR2GRAY)`.
   - Run detection: `detections = self.detector.detect(gray, estimate_tag_pose=True, camera_params=self.camera_params, tag_size=self.tag_size)`.
   - For visualization: prepare `display = bgr.copy()` if `self.imshow_enabled` else None.
   - Iterate detections; for each:
     - Compute filter status: `accepted = (d.tag_id == self.target_tag_id) and (d.hamming == 0) and (d.decision_margin >= self.decision_margin_min)`.
     - If `display is not None`:
       - Color: `(0, 255, 0)` if accepted else `(0, 0, 255)` (BGR — green/red).
       - `cv2.polylines(display, [d.corners.astype(int)], True, color, 2)`.
       - Tag id text: `cv2.putText(display, f"id={d.tag_id}", tuple(d.corners[0].astype(int) + np.array([0, -8])), cv2.FONT_HERSHEY_SIMPLEX, 0.5, color, 1, cv2.LINE_AA)`.
       - If accepted (only draw 3-axis when accepted, since pose is meaningful), use `cv2.projectPoints` with `obj_pts = np.array([[0,0,0],[0.03,0,0],[0,0.03,0],[0,0,0.03]], dtype=np.float64)`, `rvec = cv2.Rodrigues(d.pose_R)[0]`, `tvec = d.pose_t.reshape(3)`, `K = np.array([[fx,0,cx],[0,fy,cy],[0,0,1]])`, `dist = self.dist_coeffs`. Draw three lines: origin→x in red `(0,0,255)`, origin→y in green `(0,255,0)`, origin→z in blue `(255,0,0)`.
     - If NOT accepted: continue (no publish).
     - Build raw tag PoseStamped in `camera_color_optical_frame`:
       - Use `R.from_matrix(d.pose_R).as_quat()` → `[qx, qy, qz, qw]`.
       - `pose_cam = PoseStamped(); pose_cam.header.stamp = msg.header.stamp; pose_cam.header.frame_id = "camera_color_optical_frame"`.
       - `pose_cam.pose.position.x/y/z = float(d.pose_t[0]), float(d.pose_t[1]), float(d.pose_t[2])`.
       - `pose_cam.pose.orientation.x/y/z/w = float(qx), float(qy), float(qz), float(qw)`.
     - Compute target pose in camera frame by applying offset in tag-local frame:
       - `T_cam_tag = np.eye(4); T_cam_tag[:3,:3] = d.pose_R; T_cam_tag[:3,3] = d.pose_t.reshape(3)`.
       - `T_tag_target = np.eye(4); T_tag_target[:3,3] = self.offset_xyz`.
       - `T_cam_target = T_cam_tag @ T_tag_target`.
       - Build `target_pose_cam: PoseStamped` from `T_cam_target[:3,3]` (translation) and `R.from_matrix(T_cam_target[:3,:3]).as_quat()` (orientation), with the same header as `pose_cam`.
     - Transform both to output_frame; on TransformException catch + throttled warn + skip publish for this detection:
       ```python
       try:
           pose_torso = self.tf_buffer.transform(pose_cam, self.output_frame, timeout=Duration(seconds=self.tf_lookup_timeout_s))
           target_torso = self.tf_buffer.transform(target_pose_cam, self.output_frame, timeout=Duration(seconds=self.tf_lookup_timeout_s))
       except tf2_ros.TransformException as ex:
           self._warn_throttled(f"TF transform failed: {ex}")
           continue
       ```
     - Publish:
       - `self.tag_pose_pub.publish(pose_torso)`
       - `self.target_pose_pub.publish(target_torso)`
   - HUD + show (only if display not None):
     - Compute fps: `if len(self.frame_times) >= 2: fps = (len(self.frame_times)-1) / (self.frame_times[-1] - self.frame_times[0]) else: fps = 0.0`.
     - Find best displayed detection's margin (max margin across all detections, or 0 if none).
     - HUD text: `f"margin={margin:.1f}  fps={fps:.1f}  id={self.target_tag_id}"`. Draw at bottom-right with `cv2.putText`. Use a contrasting background rectangle for legibility.
     - `cv2.imshow(self.window_name, display)`.
     - `key = cv2.waitKey(1) & 0xFF`. If `key == ord('q')`: `cv2.destroyWindow(self.window_name); self.imshow_enabled = False`.

6. `_warn_throttled(self, msg: str, period_s: float = 2.0)`:
   - Use `time.monotonic()` to throttle. Maintain `self.last_warn_time`. If `now - self.last_warn_time >= period_s`: `self.get_logger().warn(msg); self.last_warn_time = now`. (Note: rclpy `get_logger().warn(..., throttle_duration_sec=...)` works on Humble but requires explicit clock; the simple manual throttle is more portable across distros.)

7. `def main(args=None)`:
   - `rclpy.init(args=args)`
   - `node = AprilTagDetectorNode()`
   - try: `rclpy.spin(node)` except KeyboardInterrupt: pass
   - finally: `if node.imshow_enabled: cv2.destroyAllWindows(); node.destroy_node(); rclpy.try_shutdown()`

8. Standard footer: `if __name__ == '__main__': main()`.

Implementation rules (deep_work_rules):
- DO NOT hardcode `tag_size`, `target_tag_id`, `decision_margin_min`, `offset_xyz`, topic names, or `output_frame`. All MUST come from `self.<param>` set in `__init__`.
- DO NOT use `cv2.WINDOW_AUTOSIZE` (use `WINDOW_NORMAL` so user can resize).
- DO NOT add extra publishers, TF broadcasters, or `/apriltag/debug_image` topics (per D-19 deferred list).
- DO NOT call `rclpy.spin_once` or use a multi-threaded executor — single-threaded default is correct.
- DO mark file executable when created (chmod +x — handled by colcon install via `install(PROGRAMS ...)` automatically; source file permissions don't strictly need +x but it's idiomatic).
</action>
