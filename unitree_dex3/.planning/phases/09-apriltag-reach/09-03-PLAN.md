    ```python
    TARGETS = [
        ('center',     0.40, -0.20,  0.00),
        ('right-side', 0.40, -0.40,  0.00),
    ```

    4. **class AprilTagReachUAT(Node):**
       - `super().__init__('apriltag_reach_uat')`
       
       **4a. Parameters:**
       - `error_threshold`: 0.03 (float, 3cm per D-23)
       - `settle_time`: 1.0 (float, seconds to wait after trajectory completion for joint_states to settle)
       - `timeout_sec`: 10.0 (float, max wait for trajectory per point)

       **4b. State variables:**
       - `self._phase = 'init'` — FSM state: 'init' -> 'waiting_tag' (per point) -> 'waiting_traj' -> 'measuring' -> 'next_point' -> 'done'
       - `self._current_index = 0` — current point index in TARGETS
       - `self._joint_state = None` — latest JointState msg
       - `self._traj_received = False` — flag for trajectory arrival
       - `self._expected_pos = None` — (x, y, z) from /apriltag/target_pose
       - `self._traj_arrival_time = None` — timestamp when trajectory was received
       - `self.results = []` — list of (label, expected, actual, error, PASS/FAIL) tuples
       - `self._exit_status = 0`
       - `self._publish_time = None` — time when we moved to waiting_traj phase
       - `self._tag_seen = False` — whether any /apriltag/target_pose has arrived since last point

       **4c. KDL FK model (built once in __init__):**
       Load URDF from `get_package_share_directory('unitree_g1_dex3_stack') + '/urdf/g1_29dof_lock_waist_with_hand_rev_1_0_collision_primitives.urdf'` (same as planner.launch.py default).

       Build Pinocchio model:
       - RIGHT_ARM_URDF_JOINTS = list of 7 right-arm joint names (same as read_tcp_pose.py)
       - Lock all other joints via `pin.buildReducedModel(model, lock_ids, np.zeros(model.nq))`
       - Add TCP frame: `pin.Frame('right_tcp', reduced.getJointId('right_wrist_yaw_joint'), pin.SE3(np.eye(3), np.array([0.175, 0.0, 0.0]).T), pin.FrameType.OP_FRAME)`
       - Cache `self._reduced`, `self._rmodel`, `self._rdata` = model.createData(), frame IDs for 'torso_link' and 'right_tcp'

       **4d. Subscriptions:**
       - `/joint_states` (sensor_msgs/JointState, QoS 10): stores latest joint state
       - `/joint_trajectory_targets` (trajectory_msgs/JointTrajectory, QoS 10): sets `_traj_received = True`, records arrival time (self.get_clock().now())
       - `/apriltag/target_pose` (geometry_msgs/PoseStamped, QoS 10): records `_expected_pos = (msg.pose.position.x, y, z)` and `_tag_seen = True`

       **4e. Timer:**
       - `self.create_timer(0.1, self._tick)` — FSM tick

       **4f. Startup log:**
       ```
       [UAT] Phase 9 End-to-End Verification Harness
       [UAT] Targets: 4 (center, right-side, low, diag)
       [UAT] Place tag at point 1 (center: 0.40, -0.20, 0.00), press G
       ```

    5. **_tick(self) FSM:**
       Dispatch on self._phase:

       **Phase 'init' (first tick only):**
       - Set `_phase = 'waiting_tag'`
       - Log current point prompt with coordinates
       - Return

       **Phase 'waiting_tag':**
       - If `_tag_seen` and `_expected_pos` is not None:
         - Log `[UAT] Tag detected at {expected_pos}`
         - Log `[UAT] Press G to trigger, or wait...`
         - Set `_phase = 'waiting_traj'`
         - Set `_traj_received = False`
         - Set `_publish_time = self.get_clock().now()`
       - Return

       **Phase 'waiting_traj':**
       - If `_traj_received`:
         - Log `[UAT] Trajectory received, waiting {self.settle_time}s for joint states to settle`
         - Set `_phase = 'measuring'`
         - Set `_traj_arrival_time = self.get_clock().now()`
         - Set `_wait_start = time.time()`
       - Else if timeout exceeded (from _publish_time):
         - Log `[UAT] Timeout waiting for trajectory at point {label}`
         - Record FAIL result with error_m=999.0
         - Set `_phase = 'next_point'`
       - Return

       **Phase 'measuring':**
       - Wait `settle_time` seconds after traj_arrival_time:
       - elapsed = (self.get_clock().now() - traj_arrival_time).nanoseconds * 1e-9
       - If elapsed >= settle_time:
         - Call `_compute_tcp_position()` to get actual TCP position
         - Compute error = euclidean distance between _expected_pos and actual
         - ok = error <= self.error_threshold (3cm)
         - Determine label from TARGETS[self._current_index]
         - Append (label, _expected_pos, actual, error, ok) to self.results
         - Log result line:
         ```
         [UAT] Point {idx+1}: expected=({exp_x:.3f}, {exp_y:.3f}, {exp_z:.3f}) actual=({act_x:.4f}, {act_y:.4f}, {act_z:.4f}) error_m={err:.4f} {'PASS' if ok else 'FAIL'}
         ```
         - Set `_phase = 'next_point'`
       - Return

       **Phase 'next_point':**
       - `_current_index += 1`
       - If `_current_index >= len(TARGETS)`:
         - Log `[UAT] All targets complete`
         - Set `_phase = 'done'`
         - Call `_summarize()`
       - Else:
         - Reset: `_tag_seen = False`, `_expected_pos = None`, `_traj_received = False`
         - Set `_phase = 'waiting_tag'`
         - Log new point prompt:
         ```
         [UAT] Place tag at point {idx+1} ({label}: {x:.2f}, {y:.2f}, {z:.2f}), press G
         ```
       - Return

       **Phase 'done':**
       - No action; _summarize already called, main loop exits on _finished flag
       - Return

    6. **_compute_tcp_position(self) -> tuple (x, y, z) in torso_link frame:**
       - Use latest self._joint_state
       - Build joint name -> position dict from _joint_state
       - Extract 7 right-arm joint values in URDF order (read_tcp_pose.py RIGHT_ARM_URDF_JOINTS order)
       - Build numpy array q of size model.nq (reduced model), zeros-filled
       - For each of the 7 right arm joints: find index in model, set q[idx] = joint_value
       - Run `pin.framesForwardKinematics(self._rmodel, self._rdata, q)`
       - Get torso_link frame pose and right_tcp frame pose
       - `tcp_in_torso = self._rdata.oMf[torso_id].actInv(self._rdata.oMf[tcp_id])`
       - Return `(tcp_in_torso.translation[0], tcp_in_torso.translation[1], tcp_in_torso.translation[2])`

    7. **_summarize(self):**
       Build and print results table:
       ```
       Point    Expected(xyz)          Actual(xyz)             Error(m)  Status
       ------------------------------------------------------------------------
       center   (0.400, -0.200, 0.000) (0.398, -0.202, 0.003)  0.0045   PASS
       right-side ...
       low ...
       diag ...
       ------------------------------------------------------------------------
       PASS_COUNT N/4
       ```
       - Set `self._exit_status = 0` if all PASS, `1` otherwise (D-24: 4/4 required for milestone sign-off)
       - Set `self._finished = True` to trigger main loop exit

    8. **main() with spin_once loop (matching adaptive_orientation_ab.py pattern):**
    ```python
    def main(args=None):
        rclpy.init(args=args)
        node = AprilTagReachUAT()
        exit_status = 1
        try:
            while rclpy.ok() and not node._finished:
                rclpy.spin_once(node, timeout_sec=0.1)
            exit_status = node._exit_status
        except KeyboardInterrupt:
            exit_status = 130
        finally:
            node.destroy_node()
            rclpy.try_shutdown()
        sys.exit(exit_status)
    ```
