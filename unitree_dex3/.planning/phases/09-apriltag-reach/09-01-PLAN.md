    4. **_target_cb(self, msg: PoseStamped):**
       - Append `(msg.pose.position.x, msg.pose.position.y, msg.pose.position.z)` to position_cache
       - Set `self._last_stamp = msg.header.stamp`
       - Set `self._last_orientation = msg.pose.orientation` (copy, D-08)
       - Set `self._last_target = msg`

    5. **_traj_cb(self, msg: JointTrajectory):**
       - If `msg.points` is empty, return (ignore empty trajectory)
       - Compute duration from last point's time_from_start + 1.0s safety margin
       - Cancel existing completion timer if one exists
       - Create a new one-shot timer with that duration
       - Timer callback sets `_waiting_for_completion = False`

    6. **_retry_shoulder_lookup(self):**

       - On success: store `(tx, ty, tz)` tuple, log INFO with cached origin
       - On any exception: log WARN with exception message
       - Implement a retry counter: increment on each attempt, if > 10 attempts log FATAL and raise SystemExit (bridge cannot operate without shoulder origin)

       ```
       if not self.position_cache:

           return

       if self._shoulder_origin is None:
           self.get_logger().warn('[apriltag_goal_bridge] shoulder origin not yet available')
           return
       ```

       **Guard 3 — in-flight (D-03):**
       ```
       if self._waiting_for_completion:
           self.get_logger().warn('[apriltag_goal_bridge] previous goal still in flight, ignoring G')
           return
       ```

       **Guard 4 — stale (D-09):**
       Compute age in seconds = (now - self._last_stamp).nanoseconds * 1e-9. If > self.stale_threshold_s:
       ```
       self.get_logger().warn(f'[apriltag_goal_bridge] no fresh AprilTag pose (last seen {age:.1f} s ago)')
       return
       ```

       **Guard 5 — reachability (D-11, D-14):**
       Compute sliding average position (pure Python sum/len, no numpy) from self.position_cache. Compute euclidean distance to self._shoulder_origin. If distance >= self.reach_max:
       ```
       self.get_logger().warn(f'[apriltag_goal_bridge] reach exceeds {dist:.3f} m > {self.reach_max} m, not publishing')
       return
       ```

       **On all guards pass (D-01):**
       - Set `self._waiting_for_completion = True`
       - Build PoseStamped: frame_id = 'torso_link', stamp = now(), position = averaged (x,y,z), orientation = self._last_orientation (copied per D-08)
       - `self.goal_pub.publish(goal)`
       - Log INFO (exact format):
       ```
       [apriltag_goal_bridge] G pressed — target=({avg_x:.3f}, {avg_y:.3f}, {avg_z:.3f}) @ torso_link, |target-shoulder|={dist:.3f} m, publishing /goal_pose
       ```
