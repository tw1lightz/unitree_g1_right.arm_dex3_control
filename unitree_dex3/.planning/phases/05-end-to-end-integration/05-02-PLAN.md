<action>
创建 `src/unitree_g1_dex3_stack-main/scripts/keyboard_trigger_node.py`，shebang `#!/usr/bin/env python3`：

1. 导入：rclpy、Node、Detection3DArray（vision_msgs.msg）、PoseStamped（geometry_msgs.msg）、os、select、termios、tty、math

2. KeyboardTriggerNode 类继承 Node：
   - 订阅 `/detections_3d`（Detection3DArray），回调缓存最新 msg 到 `self.latest_detections`
   - 发布者：`/goal_pose`（PoseStamped）
   - 键盘读取：通过 `/dev/tty` 打开文件描述符，`tty.setraw(fd)`
   - 定时器：`create_timer(0.1, self.timer_callback)` — 10Hz 轮询键盘

3. timer_callback：
   - 用 `select.select([self.fd], [], [], 0.0)` 非阻塞检查
   - 读到 'k' 或 'K' 时调用 `self.trigger()`

4. trigger() 方法：
   - 若 `self.latest_detections` 为 None 或 detections 列表为空：打印 WARN 并返回
   - 遍历 detections，计算每个 detection 的欧氏距离：`sqrt(x² + y² + z²)`（bbox.center.position）
   - 选距离最小的 detection
   - 计算目标点（D-07）：
     - `cx = det.bbox.center.position.x`
     - `cy = det.bbox.center.position.y`
     - `cz = det.bbox.center.position.z`
     - `sy = det.bbox.size.y`
     - `y_bottom = cy + sy / 2.0`
     - `y_target = y_bottom - sy * 0.1`
   - 构造 PoseStamped：
     - `header.frame_id` = 检测消息的 header.frame_id
     - `header.stamp` = self.get_clock().now().to_msg()
     - `pose.position` = (cx, y_target, cz)
     - `pose.orientation.w` = 1.0
   - 发布到 `/goal_pose`
   - 打印 INFO：`[KeyboardTrigger] K pressed — targeting nearest object at ({cx:.3f}, {y_target:.3f}, {cz:.3f}), publishing /goal_pose`

5. 节点销毁时恢复终端设置：`termios.tcsetattr(self.fd, termios.TCSADRAIN, self.old_settings)` + `os.close(self.fd)`

6. main()：rclpy.init → 创建节点 → rclpy.spin → cleanup → rclpy.shutdown
</action>
